# RPI Lab Project Structure

This repository is organized into the following main folders:

- **rf/**: Contains all existing content and scripts related to RF (radio frequency) and general Raspberry Pi setup.
- **tui/**: Dedicated to TUI (Text User Interface) development for Raspberry Pi projects. Place all TUI-related code and resources here.
- **display/**: For scripts and tools to configure and manage displays attached to Raspberry Pi devices.

Refer to each folder's README for more details as development progresses.
